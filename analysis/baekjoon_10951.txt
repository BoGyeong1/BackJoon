-문제 파악
문제
두 정수 A와 B를 입력받은 다음, A+B를 출력하는 프로그램을 작성하시오.

입력
입력은 여러 개의 테스트 케이스로 이루어져 있다.
각 테스트 케이스는 한 줄로 이루어져 있으며, 
각 줄에 A와 B가 주어진다. (0 < A, B < 10)
-유추 파악
입력에서 더이상 읽을 수 있는 데이터가 없을때 반복문을 종료하는 것
스캐너는 예외를 던짐
버퍼리더일 경우 null을 반환
-주요 단어 영문 이름 선정
입력 - input
출력 : output
더하기 : sum
-테스트 케이스
1 1 ->2
2 3 ->5
3 4 ->7
9 8 ->17
5 2 ->7
-프로그래밍 순서
1.입력
2.공백으로 분리
3. null일때 종료

-try-on
-실패 시 원인 기록
